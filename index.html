Sobre mí

Soy técnica en Relaciones Públicas y estudiante de Desarrollo Web y Aplicaciones Digitales en el Instituto Superior Santa Rosa de Calamuchita. Mi objetivo es especializarme en el desarrollo web, abarcando tanto front-end como back-end, para convertirme en una desarrolladora full stack. Me apasiona la creación de soluciones tecnológicas que mejoren la experiencia del usuario y optimicen procesos en diferentes plataformas.

Proyectos

Desarrollos Front-End

Como estudiante de desarrollo front-end, estoy aprendiendo a crear interfaces de usuario interactivas y atractivas utilizando tecnologías como HTML, CSS y JavaScript. Me estoy familiarizando con frameworks como React y herramientas de diseño responsivo para asegurar que las aplicaciones se adapten a distintos dispositivos. Además, aprendo a optimizar la experiencia del usuario, garantizando tiempos de carga rápidos y navegaciones fluidas, siempre buscando mejorar la usabilidad y la accesibilidad de los sitios web.

Desarrollos Back-End

En el área de back-end, estoy aprendiendo a manejar bases de datos, crear APIs y desarrollar la lógica que hace posible el funcionamiento de las aplicaciones. Utilizo lenguajes como Node.js y bases de datos como MySQL para gestionar el almacenamiento y procesamiento de datos. También estoy profundizando en temas de autenticación, seguridad y gestión de servidores, con el fin de construir sistemas escalables y eficientes que puedan soportar el tráfico y las necesidades de los usuarios.

Desarrollos Full stack

Mi formación en full stack me permite integrar los conocimientos de front-end y back-end para desarrollar aplicaciones web completas. Aprendo a conectar las interfaces de usuario con las bases de datos, gestionar la interacción entre el cliente y el servidor, y asegurar que todo el sistema funcione de manera armónica. Esta visión global del desarrollo me ayuda a comprender mejor cada etapa del proceso y a crear soluciones más coherentes, eficientes y optimizadas, desde la presentación hasta el manejo de datos.
