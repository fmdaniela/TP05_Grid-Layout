Durante esta práctica, utilicé JavaScript para interactuar con el DOM y crear una experiencia dinámica. Desarrollé la simulaciòn de una planta de silos que permite cargar datos y, utilizando eventos y funciones de manipulación del DOM, logré actualizar los elementos HTML en tiempo real. Como se muestra en la imagen, al cargar los datos, estos aparecen directamente en la página sin necesidad de recargarla, lo que mejora la interactividad y la fluidez del sitio web. Esta práctica consolidó mi habilidad para integrar lógica en JavaScript con la estructura HTML existente.
