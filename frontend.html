En esta práctica, desarrollé una página web desde cero utilizando HTML y CSS. Apliqué conceptos clave como la semántica HTML para estructurar el contenido y Flexbox para lograr un diseño responsivo. El sitio se adapta a diferentes tamaños de pantalla, garantizando una experiencia óptima en dispositivos móviles y de escritorio. Además, utilicé estilos personalizados para botones, encabezados y secciones, logrando una apariencia moderna y profesional. La imagen muestra el resultado final, donde se puede apreciar un diseño limpio, con colores armoniosos y tipografía clara.
