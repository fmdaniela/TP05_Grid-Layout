En este proyecto, desarrollé una tienda ecommerce que integra tanto el front-end como el back-end. Utilicé HTML, CSS y JavaScript en el cliente para crear una interfaz atractiva y responsiva, mientras que en el servidor implementé Node.js y una base de datos con MySQL. La tienda permite a los usuarios navegar por productos, agregar artículos al carrito y realizar compras simuladas. Como se aprecia en la imagen, el sistema de autenticación y gestión de inventario se implementó de manera fluida, conectando ambos lados del desarrollo full stack de forma eficiente y profesional.
